Project Description

This project is concerned with cars exceeding speed limit captured by the radar, main idea is to automate this system by implementing an application which extract license information from images taken by radar. Our point of concern is to apply the process on Arabic license plates. Our application is able to recognize the Arabic characters and numbers from images of license plates.

Project Details

  • We prepared a dataset for various kinds of car plates found in Egypt
  • We tried to cover all possible real-life conditions regarding images for the plates
  • We made a survey to know any kind of work related to ours and there is no such a project found in Egypt
  • We prepared a neat package for the source code and there is a full documentation for it
